Validate Local RichSnippet

Validate local html with Google Rich Snippets using this Bookmarklet in Firefox

This bookmarklet requires jQuery. Simply drag it to your bookmark bar and click on it to send the current page's html code to Google's Rich Snippets tool.
This is great for working with local files, instead of doing

  1. View source
  2. Select all
  3. Copy
  4. Close
  5. New tab with the Rich Snippets tool
  6. Paste & submit

Validate Local RichSnippet

You can assign it a keyword to work as a shortcut within Mozilla Firefox.

Here is the code that the bookmarklet will run:

var htmlForm='<form id="rstt-html-form"enctype="multipart/form-data"method="POST" ';
htmlForm+='target="local_validator" ';
htmlForm+='action="http://www.google.com/webmasters/tools/richsnippets/upload">';
htmlForm+='<textarea name="html"id="html-input">Paste html here.</textarea>';
htmlForm+='</form>';
$('body').append($(htmlForm));
$form=$('#rstt-html-form');
$textarea=$('#html-input');
var fullHtml=$('html').html();
$textarea.val(fullHtml);
$form.submit();
