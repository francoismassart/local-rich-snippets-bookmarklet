Validate Local RichSnippet

Validate local html with Google Rich Snippets using this Bookmarklet in Firefox

This bookmarklet requires jQuery. Simply drag it to your bookmark bar and click on it to send the current page's html code to Google's Rich Snippets tool.
This is great for working with local files, instead of doing 1. View source 2. Select all 3. Copy 4. Close 5. New tab with the Rich Snippets tool 6. Paste ;-)
